# TaskFlow Pro — Feature Documentation

## Current Features (v3.2.x)

### Task Management
- Create, edit, delete tasks with title, description, due date, priority (Low/Medium/High/Critical)
- Task statuses: To Do, In Progress, Done, Archived
- Assign tasks to team members (Premium plan)
- Drag-and-drop task reordering in list and board views
- Task attachments: images, PDFs, documents up to 10MB
- Task comments and activity log

### Projects
- Group tasks into projects with color-coded labels
- Project-level permissions: Owner, Editor, Viewer
- Project templates for common workflows
- Project progress bar based on completed tasks

### Collaboration (Premium)
- Team workspaces with shared projects
- Real-time task updates across team members
- @mentions in task comments
- Team activity feed

### Views
- List view (default)
- Kanban board view
- Calendar view (basic, shows tasks by due date)
- Dashboard with task statistics and charts

### Notifications
- Push notifications for task assignments, due dates, comments
- In-app notification center
- Email digest (daily/weekly configurable)
- Notification badge on app icon showing unread count

### Search
- Full-text search across task titles and descriptions
- Filter by project, status, assignee, date range
- Recent searches saved

### Analytics Dashboard
- Tasks completed over time (line chart)
- Time tracking per project (bar chart)
- Team member productivity (Premium)
- Overdue tasks count

### Widget
- Android home screen widget showing today's tasks
- iOS widget support via WidgetKit

## Known Limitations
- No subtask support (planned for v3.3)
- No recurring tasks (planned for v3.3)
- No calendar integration (Google Calendar, Outlook) — planned for v3.4
- No dark mode (planned for v3.3)
- No offline mode on web (mobile only)
- Attachment size limit: 10MB per file
- Search does not support fuzzy matching or typo tolerance
- No PDF export capability
- No Slack/Teams integration
- Sorting limited to date only (no sort by priority/project)
- Accessibility: partial VoiceOver/TalkBack support, several unlabelled elements

## Subscription Plans
- **Free**: Up to 5 projects, 50 tasks, no collaboration
- **Premium**: $9.99/month — Unlimited projects/tasks, collaboration, analytics, priority support
- **Enterprise**: Custom pricing — SSO, admin console, audit logs, dedicated support

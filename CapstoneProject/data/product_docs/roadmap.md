# TaskFlow Pro — Product Roadmap

## v3.2.2 (Hotfix — Target: March 2026)
- Fix Settings page crash on Android (BUG-101)
- Fix authentication failure after password reset (BUG-102)
- Renew APNS certificate for iOS push notifications (BUG-105)
- Fix dashboard slow loading with pagination (BUG-104)
- Fix startup crash on OnePlus devices (BUG-109)

- Subtask support (hierarchical tasks)
- Recurring tasks (daily, weekly, monthly, custom)
- Dark mode (system-preference aware)
- Fix data sync race condition (BUG-103)
- Fix file attachment memory issue (BUG-106)
- Fix notification badge drift (BUG-108)
- Improved accessibility (VoiceOver/TalkBack audit fixes)
- Offline mode improvements

- Google Calendar integration (two-way sync)
- Outlook Calendar integration (two-way sync)
- Slack integration (notifications + task creation from messages)
- Microsoft Teams integration

- PDF export for tasks and project reports
- Advanced sorting (by priority, project, assignee, custom fields)
- Custom filter views (save and share)
- Siri Shortcuts support (iOS)
- Google Assistant support (Android)
